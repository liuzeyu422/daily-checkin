# Zeyu 的每日打卡网页

这是一个为你专属设计的每日打卡网页系统，包含如下功能：

- 每日任务显示
- 打卡按钮记录完成状态（本地存储）
- 今日完成率统计
- 一键重置按钮

## 🛠️ 部署步骤（GitHub Pages）

1. 注册 GitHub 账号并登录
2. 新建一个仓库，比如叫 `daily-checkin`
3. 上传此项目的所有文件（`index.html`, `README.md`）
4. 进入仓库 → Settings → Pages
   - Source 选择 `main` 分支，路径选择 `/ (root)`
   - 保存后几秒内会生成你的网站链接

🎉 完成！访问你的打卡网页并开始使用吧！
